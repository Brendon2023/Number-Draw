# 🎲 Number Draw

Projeto desenvolvido durante meus estudos na plataforma Alura com o objetivo de praticar lógica de programação utilizando JavaScript, HTML e CSS.

## 📖 Sobre o projeto

O **Number Draw** é uma aplicação web que realiza o sorteio de números aleatórios dentro de um intervalo definido pelo usuário, permitindo configurar:

- Número inicial
- Número final
- Quantidade de números sorteados

Além do sorteio, o projeto possui validações para evitar entradas inválidas e utiliza manipulação do DOM para atualizar a interface em tempo real.
